# Safety Notes

This project is a model rocket launch controller and should only be used in a safe, supervised launch environment.

## Intended Use

* Supervised model rocketry
* Educational hardware design
* Launch controller enclosure and safety-control design
* Use with proper launch procedures and range rules

## Safety Rules

* Follow the NAR Model Rocket Safety Code.
* Use adult supervision when required.
* Keep the launch area clear before arming.
* Do not handle igniters or rockets while the controller is armed.
* Keep the safety key removed when the controller is not being used.
* Disconnect power before working on the controller.
* Only use the controller in an appropriate outdoor launch area.

## Design Safety Features

| Feature             | Safety Purpose                                 |
| ------------------- | ---------------------------------------------- |
| Key-lock switch     | Prevents casual or accidental arming           |
| Flip-cover switch   | Makes the arming step more intentional         |
| Momentary button    | Requires active operator input                 |
| Continuity LED      | Gives connection feedback before launch        |
| Buzzer              | Warns that the controller is in an armed state |
| Large launch button | Makes the final action clear and deliberate    |

## Warning

This project is not a toy. It is intended for responsible model rocketry use only.
